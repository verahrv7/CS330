# CS 330 Portfolio Reflection

## How do I approach designing software?

Working on this project taught me to plan before I start coding. I first thought about what I wanted my scene to look like and then broke it into smaller objects that were easier to create. As I worked through each milestone, I adjusted my design based on what worked and what needed improvement.

## How do I approach developing programs?

I learned that building a project step by step makes it much easier to manage. Instead of trying to complete everything at once, I added features one at a time and tested them before moving on. Throughout the milestones, I also reorganized my code into separate classes, which made it easier to read and update.

## How can computer science help me in reaching my goals?

This course gave me experience with computational graphics, object-oriented programming, and problem-solving. These skills will help me in future computer science courses and can also be useful in careers involving software development, visualization, simulations, or interactive applications. Even if I do not work directly with OpenGL, I gained experience writing organized code and breaking large projects into smaller tasks, which are valuable skills in any programming job.
